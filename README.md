# 🧠 Автономний ШІ (MARL) в Unreal Engine 5

Цей проєкт демонструє використання алгоритмів **Continuous Multi-Agent Reinforcement Learning (MARL)** для навчання ботів у 3D-середовищі. Штучний інтелект керує "Мисливцем" (Seeker), який вчиться самостійно знаходити та переслідувати ціль ("Хайдера"), використовуючи дані з віртуальних сенсорів (радар, промені дистанції).

"Мозок" ШІ написаний на Python за допомогою бібліотеки **PyTorch** і спілкується з ігровим рушієм Unreal Engine через TCP-сокети в режимі реального часу.

---

## 📂 Структура проєкту

* **`AI_Training/`** — Python-сервер, що містить логіку нейромережі, систему нагород (Reward System) та цикл навчання. Головний файл запуску: `AI_Training/server.py`.
* **`Kursova2/`** — Вихідні файли проєкту Unreal Engine 5 (Блюпринти, C++ класи для сокетів, 3D-сцена). Головний файл проєкту: `Kursova2/Kursova2.uproject`.
* **`Models/`** — Резервні копії навчених моделей (файли `.pth`).
* **`Windows/`** — *(Видалено з файлів коду через розмір)* Готова скомпільована гра знаходиться у розділі **Releases**.

---

## 🚀 Як запустити готову гру та ШІ (Режим демонстрації)

Щоб побачити, як ШІ працює в реальному часі, вам не обов'язково встановлювати Unreal Engine. Достатньо запустити Python-сервер та завантажити скомпільовану гру.

### 1. Підготовка Python-середовища
Переконайтеся, що у вас встановлено Python 3.8 або новіше. Відкрийте термінал і встановіть необхідну бібліотеку для роботи нейромережі:
```bash
pip install torch
```

### 2. Запуск "Мозку" (Python-сервер)
Щоб ШІ почав думати, потрібно запустити сервер. Відкрийте термінал у кореневій папці проєкту та виконайте команду:
```bash
python AI_Training/server.py
```
*Сервер успішно запущено, якщо в консолі з'явилося повідомлення: `Continuous MARL Server listening on 127.0.0.1:5555`. Тепер він чекає на підключення гри.*

### 3. Запуск Гри (з розділу Releases)
1. Перейдіть до розділу **[Releases](https://github.com/N0RMY/Unreal-MARL-AI/releases)**.
2. Завантажте архів з останньою версією гри (файл `.zip`).
3. Зайдіть у розпаковану папку `Windows` та запустіть файл **`Kursova2.exe`**.
4. Гра автоматично підключиться до Python-сервера. Перейдіть у вікно терміналу — там ви побачите потік даних, координати та процес прийняття рішень ШІ в реальному часі.

## 🛠️ Як відкрити проєкт в Unreal Engine 5

Якщо ви хочете подивитися, як налаштована логіка ботів, сенсори або змінити карту, ви можете відкрити вихідний проєкт.

**Вимоги:** Unreal Engine версії 5.x та встановлений компілятор C++ (Visual Studio).

1. Перейдіть до папки **`Kursova2/`**.
2. Знайдіть файл проєкту **`Kursova2/Kursova2.uproject`** і двічі клацніть на нього.
3. Оскільки проєкт використовує кастомний C++ клас (`SocketClient`), рушій може запитати: *"Would you like to rebuild them now?"* (Чи бажаєте ви перекомпілювати модулі?). Натисніть **Yes**.
4. Після компіляції відкриється редактор. Основна логіка ботів знаходиться в папці `Content/Blueprints` (файли `BP_Seeker` та `BP_Hider`).

---

## 🧠 Процес навчання ШІ

Модель використовує підхід **Policy Gradient**. За кожну корисну дію (наближення до цілі, правильний поворот камери) бот отримує позитивну нагороду, а за шкідливу (зіткнення зі стінами, втрата часу, рух задом) — штраф. 

Під час роботи гри Python-сервер автоматично зберігає прогрес і оновлює файли ваг (`seeker_model.pth` та `hider_model.pth`) кожні 50 раундів.
